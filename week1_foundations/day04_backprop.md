# Day 4：反向传播算法

## 网络结构

```
x → [W1,b1] → z1 → ReLU → a1 → [W2,b2] → z2 → sigmoid → ŷ → L
```

目标：算出 L 对每个参数（W1, b1, W2, b2）的梯度，然后梯度下降更新。

---

## 为什么需要激活函数

如果没有激活函数，两层线性变换叠在一起：

```
z2 = W2·(W1·x + b1) + b2 = (W2·W1)·x + (W2·b1 + b2) = W'·x + b'
```

无论叠多少层，结果永远等价于一个单层线性模型。  
**激活函数打破线性**，让网络真正获得逐层叠加的表达能力。

ReLU 的效果：

```python
relu(z) = max(0, z)   # 负数截断为 0，非线性
```

没有 ReLU，XOR 这种线性不可分问题永远无法解决。

---

## 前向传播

```
z1 = W1 · x + b1        # 线性变换
a1 = ReLU(z1)           # 激活
z2 = W2 · a1 + b2       # 线性变换
ŷ  = sigmoid(z2)        # 输出概率
L  = -[y·log(ŷ) + (1-y)·log(1-ŷ)]   # 交叉熵损失
```

> 前向传播时要缓存 x, z1, a1，反向传播要用。

---

## 反向传播：五步链式法则

链式法则核心思想：**上游梯度 × 当前操作的局部导数**

### Step 1：∂L/∂z2 = ŷ - y

sigmoid + 交叉熵联合求导，推导过程：

```
∂L/∂ŷ  = -y/ŷ + (1-y)/(1-ŷ)
∂ŷ/∂z2 = ŷ·(1-ŷ)        ← sigmoid 导数

∂L/∂z2 = ∂L/∂ŷ · ∂ŷ/∂z2
        = [-y/ŷ + (1-y)/(1-ŷ)] · ŷ·(1-ŷ)
        = -y·(1-ŷ) + (1-y)·ŷ
        = ŷ - y
```

结果极其简洁：**预测值 - 真实值**。预测越偏，梯度越大，纠正力度越大。

### Step 2：∂L/∂W2 和 ∂L/∂b2

```
z2 = W2 · a1 + b2

∂L/∂W2 = ∂L/∂z2 · a1ᵀ = (ŷ-y) · a1ᵀ   # 外积，形状和 W2 一致
∂L/∂b2 = ∂L/∂z2 = (ŷ-y)               # 偏置梯度 = 上游梯度
```

**规律**：`∂L/∂W = 上游梯度 · 输入ᵀ`，这对所有线性层成立。

### Step 3：∂L/∂a1

```
∂L/∂a1 = W2ᵀ · ∂L/∂z2
```

梯度沿权重矩阵的**转置**往回传。前向时 `z2 = W2 · a1`，反向时用 W2ᵀ，这是线性映射的基本性质。

### Step 4：∂L/∂z1（穿过 ReLU）

```
ReLU'(z) = 1 if z > 0 else 0

∂L/∂z1 = ∂L/∂a1 ⊙ ReLU'(z1)    # ⊙ 逐元素乘
```

ReLU 的梯度是开关：正数区域放行，负数区域截断。  
注意用 z1（激活前的值），不是 a1。

### Step 5：∂L/∂W1 和 ∂L/∂b1

```
∂L/∂W1 = ∂L/∂z1 · xᵀ    # 和 Step 2 完全对称
∂L/∂b1 = ∂L/∂z1
```

---

## 梯度下降更新

```
W ← W - α · ∂L/∂W
```

- 梯度指向 L 增大最快的方向
- 减去梯度 = 往损失减小的方向走
- α 是学习率：太小收敛慢，太大震荡发散

---

## 梯度验证（有限差分法）

```
∂L/∂W ≈ [L(W+ε) - L(W-ε)] / 2ε
```

不用推导，直接用定义逼近。和手推梯度对比，差值 < 1e-5 说明推导正确。这也是 PyTorch 内部测试 autograd 的方法，叫 **gradient check**。

---

## 代码对照

| 公式 | 代码 |
|------|------|
| `dz2 = (ŷ-y) / batch` | `dz2 = (y_hat - y) / batch` |
| `dW2 = dz2 · a1ᵀ` | `dW2 = dz2 @ a1.T` |
| `db2 = dz2` | `db2 = np.sum(dz2, axis=1, keepdims=True)` |
| `da1 = W2ᵀ · dz2` | `da1 = W2.T @ dz2` |
| `dz1 = da1 ⊙ ReLU'(z1)` | `dz1 = da1 * relu_grad(z1)` |
| `dW1 = dz1 · xᵀ` | `dW1 = dz1 @ x.T` |
| `W ← W - α·dW` | `W1 = W1 - lr * dW1` |

> `db2` 要用 `np.sum(axis=1)` 是因为 loss 取的是均值，偏置对每个样本都生效，梯度沿 batch 维度求和。
