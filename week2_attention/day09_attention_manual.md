# Day 9：手算注意力权重矩阵

## 完整流程：从 X 到输出

```
X (seq, d_model)
    ↓ 三个投影矩阵
Q = X·Wq    K = X·Wk    V = X·Wv
    ↓
scores = Q·Kᵀ / √dₖ
    ↓
weights = softmax(scores)
    ↓
output = weights · V
```

---

## 手算例子：4 个 token

句子：`"the cat sat mat"`  
设定：d_model=2，d_k=2，Wq=Wk=Wv=I（单位矩阵，简化计算）

**输入 X：**
```
the: [1, 0]
cat: [0, 1]
sat: [1, 1]
mat: [0, 2]
```

**Step 1：Q·Kᵀ（手算每一格）**

每个格子 = 对应 query 行 · key 行 的点积：

```
          k_the  k_cat  k_sat  k_mat
q_the  [  1·1+0·0  1·0+0·1  1·1+0·1  1·0+0·2 ]   = [1, 0, 1, 0]
q_cat  [  0·1+1·0  0·0+1·1  0·1+1·1  0·0+1·2 ]   = [0, 1, 1, 2]
q_sat  [  1·1+1·0  1·0+1·1  1·1+1·1  1·0+1·2 ]   = [1, 1, 2, 2]
q_mat  [  0·1+2·0  0·0+2·1  0·1+2·1  0·0+2·2 ]   = [0, 2, 2, 4]
```

**Step 2：除以 √dₖ = √2 ≈ 1.414**

```
[[0.71, 0.00, 0.71, 0.00],
 [0.00, 0.71, 0.71, 1.41],
 [0.71, 0.71, 1.41, 1.41],
 [0.00, 1.41, 1.41, 2.83]]
```

**Step 3：softmax 每一行（手算第 0 行）**

```
原始: [0.71, 0.00, 0.71, 0.00]
减最大值: [0.00, -0.71, 0.00, -0.71]
exp: [1.00, 0.49, 1.00, 0.49]
÷ 总和(2.98): [0.336, 0.164, 0.336, 0.164]
```

所以 `the` 最关注自己和 `sat`（点积相同，权重相同）。

**Step 4：weights × V**

以 `cat`（第 1 行）为例：
```
output[cat] = 0.164×[1,0] + 0.455×[0,1] + 0.237×[1,1] + 0.144×[0,2]
            = [0.401, 1.027]
```

这个输出向量已经融合了 `mat`（权重 0.455 最高）的信息，不再是 cat 的原始 embedding。

---

## 关键观察

**`cat` 最关注 `mat`，不是自己**  
因为 `q_cat=[0,1]`，`k_mat=[0,2]`，点积=2，比 `q_cat·k_cat=1` 大。  
注意力模式完全由 Q/K 向量的方向决定，而不是 token 的位置。

**`mat` 自注意力权重最高**  
`q_mat=[0,2]`，`k_mat=[0,2]`，点积=4，是所有配对里最大的。  
很多 token 把最大权重给自己，这在实际模型里很常见。
